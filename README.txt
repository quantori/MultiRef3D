# MultiRef3D
This repository hosts files needed to demonstrate and reproduce results presented in the manuscript "Multi-reference poly-conformational computational methods for de-novo design, optimization, and repositioning of pharmaceutical compounds" by V.Alexandrov, A.Kirpich and Y.Gankin

The code was tested under Python 3.7 environment requiring to install the following packages (via conda install):
1. RDkit
2. ODDT
